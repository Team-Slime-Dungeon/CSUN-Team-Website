Technology

Godot Game Engine

Our game proudly uses the free, open source game engine Godot. Godot is a free game engine that uses GDScript and C#.

Godot is capable of 2D and 3D visuals and is available across many operating systems and hardware.

Godot Site: https://godotengine.org/

GDScript

Godot's built in scripting language GDScript allows for easy creation and implentation of new features.

Combining the ease of GDScript with Godot's node hierarchy system allows for a powerful tool for making game building easy.

Godot Documentation: https://docs.godotengine.org/en/stable/

Aseprite

Aseprite is a lightweight, easy to use sprite making application.

Supported on Windows, Linux and Mac, this application allowed our team to easily draw beautiful sprite art and assets and test animations.

Aseprite Site: https://www.aseprite.org/

GitHub

Our project is hosted on GitHub (including this site!)

GitHub is the world's leading version control website. GitHub's ease of use for collaboration with multiple users allowed us to easily update anything in our game instantly so that everyone on the team could stay up to date. It also hosts and manages all of the branches for the game, in case we need to revert to an older version or create a release.

Our GitHub Page: https://github.com/Team-Slime-Dungeon/Roguelike-Dungeon-Crawler
